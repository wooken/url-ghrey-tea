# url-ghrey-tea
Neovim plugin to open current view on GitHub

## Requirements
- [neovim](https://github.com/neovim/neovim)
- [pynvim](https://github.com/neovim/pynvim)
- Python 3.6+
- xdg-open

## Commands
| Command | Description |
| --- | --- |
| GHGetUrl | Copy URL to clipboard |
| GHOpenUrl | Open URL in browser |
